1 请给出 T(n) 尽可能紧凑的渐进上界并予以说明，可以假定 n 是 2 的整数次幂。(每小题 3 分，共 21 分)


  1. T(1) = T(2) = 1
    T(n) = T(n − 2) + 1 if n > 2

答：

  • 当 n 为奇数时，

    因此， $ T(n) = \frac{n+1}{2}\ ,\ n \geq 1 $


  • 同理，当 n 为偶数时

    因此， $ T(n) = \frac{n}{2}\ ,\ n \geq 1 $


  • 综上， $\exist c_1 = \frac{1}{2}, c_2 = 1, n_0 = 1, 使得 \forall n \geq n_0, c_1 · n \leq T(n) \leq c_2 · n， 即 T(n) = \Theta (n)$


  • 因此，渐进上界为: $\ \ T(n) = O(n)$


  1. T(1) = 1
    T(n) = T(n/2) + 1 if n > 1

答:

  • 取 $ a = 1, b = 2, f(n) = 1 $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = 0, 则 f(n) = \Theta(n^{log{b}a}) $, 满足主定理条件二，因此由主定理得： $ T(n) = \Theta(n^{log_{b}a}logn) = \Theta(logn) $


  • 因此，渐进上界为: $\ \ T(n) = O(logn)$


  1. T(1) = 1
    T(n) = T(n/2) + n if n > 1

答：

  • 取 $ a = 1, b = 2, f(n) = n $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = 0 < 1, 则 f(n) = \Omega(n^{log{b}a + \epsilon}) $, 满足主定理条件一，因此由主定理得： $ T(n) = \Theta(f(n)) = \Theta(n) $


  • 因此，渐进上界为: $\ \ T(n) = O(n)$


  1. T(1) = 1
    T(n) = 2T(n/2) + 1 if n > 1

答：

  • 取 $ a = 2, b = 2, f(n) = 1 $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = 1 > 0, 则 f(n) = O(n^{log{b}a - \epsilon}) $, 满足主定理条件三，因此由主定理得： $ T(n) = \Theta从) = \Theta(n) $


  • 因此，渐进上界为: $\ \ T(n) = O(n)$


  1. T(1) = 1
    T(n) = 4T(n/2) + 1 if n > 1

答：

  • 取 $ a = 4, b = 2, f(n) = 1 $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = 2 > 0, 则 f(n) = O(n^{log{b}a - \epsilon}) $, 满足主定理条件三，因此由主定理得： $ T(n) = \Theta(n^{log_{b}a}) = \Theta(n^2) $


  • 因此，渐进上界为: $\ \ T(n) = O(n^2)$


  1. T(1) = 1
    T(n) = T(n/2) + log n if n > 1

答：

  • 取 $ a = 1, b = 2, f(n) = logn $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = 0, 取 k = 1, 则 f(n) = \Theta(n^{log{b}a}log^{k}n) $, 满足主定理（拓展形式）条件二，因此由主定理（拓展形式）得： $ T(n) = \Theta(n^{log_{b}a}log^{k+1}n) = \Theta(log^{2}n) $


  • 因此，渐进上界为: $\ \ T(n) = O(log^{2}n)$


  1. T(1) = 1
    T(n) = 3T(n/2) + n2 if n > 1

答：

  • 取 $ a = 3, b = 2, f(n) = n^2 $, 则递推式可化为 $ T(n) = a \cdot T(\frac{n}{b}) + f(n) $


  • 且 $ log{b}a = log{2}3 < 2, 则 f(n) = \Omega(n^{log_{b}a + \epsilon}) $, 满足主定理条件一，因此由主定理得： $ T(n) = \Theta(f(n)) = \Theta(n^2) $


  • 因此，渐进上界为: $\ \ T(n) = O(n^2)$

2 寻找中位数问题 (19 分)

  一组数据 x = (x1, · · · , xn) 从小到大排序后的序列为 x1′, · · · , xn′，则这组数据的 中位数 M 为

  给定两个长度分别为 n、m 的有序数组 A[1..n], B[1..m]（A 与 B 均已按从小到大排序）。请设计尽可能高效的算法，求出这两个数组中所有数据的中位数，并分析其时间复杂度。

答：

  • 题目分析：
      寻找中位数即为求A、B的合数组的第 k 大元素的问题，当(m+n)为奇数时，k 为 $\frac{m+n+1}{2}$，当(m+n)为偶数时，k 为 $\frac{m+n}{2}$ 和 $(\frac{m+n}{2}+1)$，中位数为二数之平均
      因此，只需要不断将 k 一分为二，在A、B中分别找到 $\frac{k}{2}(向下取整)$的数然后比较两者的大小，较小者的数组前部被抛弃（因为不可能包含目标数），缩减k的范围，继续递归即可得到第k大的值

  • 算法实现：

  • 时间复杂度分析：
      由于每次k均缩减 $\frac{1}{2}$，而最开始 k = $\frac{m+n+1}{2}$ 或 $\frac{m+n}{2}$ 或 $(\frac{m+n}{2}+1)$；因此时间复杂度为 $O(log(m + n))$

3 双调序列最大值问题 (20 分)

若一个序列 a[1…n] 满足以下两个性质之一，则该序列称为双调序列：

  • 性质 1：存在 k ∈ [1, n]，使得 a1 ≤ a2 ≤ … ≤ ak-1 ≤ ak ≥ ak+1 ≥ … ≥ an-1 ≥ an

  • 性质 2：序列经循环移位后满足性质 1

例如，序列 [1, 2, 4, 8, 7, 5, 3] 为双调序列（满足性质 1），序列 [5, 3, 1, 2, 4, 8, 7] 也为双调序列（循环移位后可得序列 [1, 2, 4, 8, 7, 5, 3]，满足性质 2）。

给定一个双调序列 a[1…n]，请设计一个算法来求出这个序列中的最大值，并对该算法的复杂度进行分析。

答：

  • 题目分析：
      序列经循环移位后满足性质 1，即为将序列首尾相接成环之后，存在一个最大值和其两侧的两个递减序列，最后在环的某处相交；任取环中的两个元素 $a_i, a_j$, 取其中的较大值，此处设较大值为 $a_i$，则 $a_i$ 不为最小值，且有：

    • 若 $ a{(i + 1) \% n} < a_i, a_i > a{(i - 1 + n) \% n} $， 则 $a_i$ 为所求最大值
    • 若 $ a{(i + 1) \% n} > a_i > a{(i - 1 + n) \% n}$，则最大值在$ a{(i + 1) \% n}, a{(i + 2) \% n}, …, a_j$ 中
    • 若 $ a{(i + 1) \% n} < a_i < a{(i - 1 + n) \% n}$，则最大值在$ a{(i - 1 + n) \% n}, a{(i - 2 + n) \% n}, …, a_j$ 中
    • 则，通过递归不断将环二分最后即可得最大值

  • 算法实现：

  • 时间复杂度分析：
      由于每次递归数组搜索规模均至少减半，因此时间复杂度为 $O(logn)$

4 字符串等价关系判定问题 (20 分)

  给定两个长度为 n 的字符串 A 和 B，若称 A 与 B 是等价的，当且仅当它们满足如下关系之一：

  1. A 和 B 完全相同;

  2. 若将把 A 分成长度相等的两段 A1 和 A2，也将 B 分成长度相等的两段 B1 和 B2。且他们之间满足如下两种关系之一：
    a. A1 和 B1 等价且 A2 和 B2 等价;
    b. A1 和 B2 等价且 A2 和 B1 等价。

  请你设计一个高效的算法来判断两个字符串是否等价并分析你的算法的时间复杂度。

  例如，假设 A 字符串为 aaba，B 字符串为 abaa。A 字符串首先分裂为 A1 = aa，A2 = ba。 B 字符串分裂为 B1 = ab，B2 = aa。A2 字符串再分裂为 A21 = b，A22 = a。B1 字符串再分裂为 B11 = a，B12 = b。此时 A21 和 B12 相等，A22 和 B11 相等，因此根据等价关系的判断条件 2.b 可知，A2 和 B1 等价。又因为 A1 和 B2 完全相同，根据等价关系的判断条件 1 可知，A1 和 B2 等价。再次利用等价关系的判断条件 2.b，可以得出结论，字符串 A 和 B 等价。

答：

  • 题目分析：
      判断两字符串等价可以将两个字符串分别递归分至最细粒度，然后再按字典序进行归并拼接，若是归并所得的两个字符串完全相同，则由上述规则说明A、B字符串可以等价为同一个字符串，反之则不可能等价为同一个字符串

  • 算法实现：

  • 时间复杂度分析：
      每一次递归都将字符串一分为二，合并过程为字符串的比较及拼接，时间复杂度为 O(n)，因此有递归式： T(n) = 2T(n/2) + O(n)，由主定理法可解出算法的时间复杂度为 O(nlog n)

5 向量的最小和问题 (20 分)

给定 n 个二维向量 v1, v2, . . . , vn。每一个向量 vi = (xi, yi) 都可以变换为如下四种形式：
  1. $vi^1$ = (xi, yi)
  2. $vi^2$ = (−xi, yi)
  3. $vi^3$ = (xi, −yi)
  4. $vi^4$ = (−xi, −yi)
请你设计一个高效的算法从 n 个向量中找出两个向量，使得他们以某种形式相加后的模长最小。换言之，请找出两个向量 vi, vj (1 ≤ i, j ≤ n 且 i ̸= j)，以及两个整数 k1, k2(1 ≤ k1, k2 ≤ 4)，使得 ||vik1 + vjk2||2 最小。此外，请分析该算法的时间复杂度。

答：

  • 题目分析：
      分析任意两向量 $(x1, y1)，(x2, y2)$，不难证明通过变换后两者相加的模长最小可等价为求 $\sqrt{(|x1| − |x2|)^2 + (|y1| − |y2|)^2}$ 的最小值，即求向量 (|x1|, |y1|) 和 (|x2|, |y2|) 两点间的距离；因此问题可转化为：将所有向量的 x 和 y 取绝对值后（即全部变换为第一象限后），求其中距离最短的两个点的距离
      首先将所有点对 x 最彪排序，然后每次按照 x 进行划分即递归，两边递归求出一个最短距离 disLeft, disRight,那么所有点中最近的两点间距离至少为dis = min(disLeft, disRight)
      两个子集合并时，对某一点 point, 可以画一个以 point 为中心，边长为 2d 的正方形，仅需考虑在该正方形内部的点集；则该正方形内最多存在 6 个点，使之与point的距离小于 dis 因此可以将这些点按照 y 排序后选择距离 point 上下分别最近的 6 个点来比较，按y排序可在合并使用对y的归并算法可以缩减算法复杂度

  • 算法实现：

  • 时间复杂度分析：

    • 最开始的归并排序的时间复杂度为 $O(nlogn)$
    • 递归的时间复杂度满足递推式 $T(n) = 2T(\frac{n}{2}) + O(n)$，由主定理可解得复杂度为 $O(nlogn)$
    • 最后求得向量以及变换形式 $v1, v2, k1, k2$ 的时间复杂度为 $O(1)$
    • 因此总时间复杂度为 $O(nlogn)$
